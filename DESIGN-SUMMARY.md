# Design summary

- Design concept: Labor marketplace
- Layout personality: direct response
- Typography: Archivo Narrow headings with Open Sans body text
- Color palette: primary #7f1d1d, secondary #fca5a5, accent #2563eb, background #fff7ed, text #1c1917, muted #78716c
- Shape language: Pill-shaped elements
- Visual motif: worker rosters and task tickets
- Unique layout choices: first sections use Labor roster, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
